# 接口设计说明书（API）

> 状态：已充实（规范 + 现有接口 + 新增约定）。

## 1. 通用规范
- 协议：HTTPS（生产）/ HTTP（开发）；风格：RESTful JSON。
- 前缀：所有接口挂在 `/api` 下（FastAPI `include_router(prefix="/api")`）。
- 鉴权：请求头 `Authorization: Bearer <token>`（拦截器自动附加）。
- 时间：ISO 8601（`YYYY-MM-DD` / `YYYY-MM-DDTHH:MM:SS`）。
- 统一响应：`{ "code": 0, "message": "ok", "data": ... }`（待后端统一封装）。
