## 六、工程实践

1. **布局规范**：账簿类页统一采用「左侧 220px 面板 + 拖拽条 + 右侧表格区」；序时账采用全宽表格。
2. **类型安全**：TypeScript 严格模式，`vue-tsc` 类型检查 + `vite build` 双重校验。
3. **Mock 数据**：所有表格采用随机生成的借贷平衡 Mock 数据，保证演示效果。
4. **预览机制**：`python3 -m http.server 8137` 静态服务 + preview 技能推送对外 URL。
5. **版本管理**：每次功能提交独立 commit，结构清晰可追溯。
