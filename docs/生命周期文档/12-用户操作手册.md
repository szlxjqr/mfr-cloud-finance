# 用户操作手册

> 状态：已充实（菜单导航 + 操作指引骨架）。

## 1. 登录与首页
- 访问系统进入「首页」仪表盘，查看经营概览、资金、税费与营收趋势。
- 左侧为一级导航菜单，点击展开子模块。

## 2. 菜单导航总览

| 一级 | 常用入口 |
|---|---|
| 合同管理 | 人事合同 / 销售合同 / 采购合同 / 往来单位 |
| 税务管理 | 税务工作台 / 个税 / 印花税 / 发票税务汇总 |
| 报销管理 | 报销申请 / 报销审批 / 报销台账 |
| 工资管理 | 员工基本信息 / 工资列表 / 个税报表 |
| 资产管理 | 固定资产管理 |
| 账务管理 | 总账 / 发票 / 出纳 / 报表 / 结账 |
| 设置 | 科目 / 期初 / 辅助字典 / 币别 等 |

## 3. 典型操作流程（骨架）

### 3.1 新建销售合同
合同管理 → 销售合同 → 新建 → 选择客户（往来单位）→ 填写金额/税率/期限 → 提交 → 系统自动计提印花税（税务管理·印花税可查）。

### 3.2 费用报销
报销管理 → 报销申请 → 新建（填费用明细）→ 提交 → 报销审批 → 审批通过 → 报销台账生成，可下钻生成费用发票与出纳付款单。

### 3.3 查看税务汇总
税务管理 → 税务工作台：查看各税种应缴/已缴/未缴与申报状态；发票税务汇总：查看进/销项税额汇聚。
