# 需求规格说明书（SRS）

> 状态：已充实（基于现有系统 + 新增模块规划）。

## 1. 系统概述
「智慧经营」面向中小企业，将日常经营场景（合同/税务/报销/工资/资产）与账务管理（总账/发票/出纳/报表/结账）统一在一个系统中，实现业财数据贯通。

## 3. 新增需求

### 3.1 合同管理
- 人事合同：合同列表（增删改查/到期提醒/续签预警）、合同模板。
- 销售合同：合同列表（执行跟踪，关联客户、销项发票、回款）、合同模板、客户管理。
- 采购合同：合同列表（执行跟踪，关联供应商、进项发票、付款）、合同模板、供应商管理。

### 3.2 税务管理
- 税务工作台：各税种（增值税/所得税/个税/印花税）应缴·已缴·未缴 + 申报状态总览。
- 个税：计算/代扣（链工资）、申报跟踪。
- 印花税：由合同签立自动计提、申报跟踪。
- 发票税务汇总：进/销项税额汇聚，测算增值税。

### 3.3 报销管理
- 报销申请（我的申请/新建/含费用明细）。
- 报销审批（待审/已审，审批流状态机）。
- 报销台账（已通过记录，挂账/付款状态，可下钻生成费用发票+付款单）。

### 3.4 归并模块
- 工资管理、资产管理、账务管理：将上述现有模块按统一一级骨架归并，按表结构补实后端。

## 5. 非功能需求
- 性能：列表查询 < 1s（千级数据）；报表生成 < 3s。
- 安全：登录鉴权（Bearer Token）、敏感字段（身份证）脱敏存储。
- 合规：财税数据留存、审计接口文件导出（现有设置已预留）。
- 兼容：现代浏览器（Chrome/Edge）。

## 6. 数据需求（关键实体）
合同（人事/销售/采购）、往来单位、报销单及明细、税款台账。详见 06 数据库设计说明书。
